# poo.js

```javascript
npm install poo
```

### Usage
##### Require the module
```javascript
var poo = require('poo');
```

##### In the browser
```html
<script type="text/javascript" src="./poo.js"></script>
```

##### Basic Use
```javascript
poo();
```

##### Additional Functionality
```javascript
poo.thesaurus();
```
